Для компіляції та запуску високопродуктивної моделі wormhole МнК з агрегацією каналів та довільним простровим розподіленням трафіку необхідно:
1.	Встановити програму Cygwin версії 1.7;
2.	Встановити SystemC версії 2.2;
3.	Відкрити консоль Сygwin і перейти в директорію, що містить вихідний код моделі;
4.	Якщо є необхідність, змінити параметри моделі в файлі “parameters.h”;
5.  В файлі “parameters.h” встановити значення параметру optimization_mode рівним false;
6.  Налаштувати файли “destinations.h” та “links.h” відповідно до мережі, що моделюється;
7.	Для компіляції моделі виконати з консолі Сygwin команду:
g++ -I /usr/local/systemc-2.2/include -L /usr/local/systemc-2.2/lib-cygwin main.cpp -lsystemc -o run.x
В результаті компіляції буде створено виконавчий файл “run.x”; 
8.	Для початку процедури моделювання необхідно запустити на виконання файл “run.x”. Результати моделювання будуть виведені в консоль Сygwin і за 
необхідності можуть бути перенаправлені в файл.

Для компіляції та запуску програмної реалізації методу параметричного налаштування wormhole МнК з агрегацією каналів (АК) необхідно:
1.	Встановити програму Cygwin версії 1.7;
2.	Встановити SystemC версії 2.2;
3.	Відкрити консоль Сygwin і перейти в директорію, що містить вихідний код високопродуктивної моделі МнК з АК;
4.	Встановити в файлах “parameters.h” і “destinations.h” параметри МнК, для якої буде виконуватись процедура параметричного налаштування;
5.	Встановити в файлі “parameters.h” значення максимально допустимої транспортної затримки для потоків даних в МнК і максимально допустиму кількість 
фізичних каналів (ФК) в транках;
6.	В файлі “parameters.h” встановити значення параметру optimization_mode рівним true;
7.	Скомпілювати високопродуктивну модель спеціалізованої wormhole МнК з АК, для якої буде виконуватись процедура парметричного налаштування, 
виконавши з консолі Cygwin команду:
g++ -I /usr/local/systemc-2.2/include -L /usr/local/systemc-2.2/lib-cygwin main.cpp -lsystemc -o run.x
8.	Скомпілювати програму, що реалізує метод параметричного налаштування wormhole МнК з АК, виконавши з консолі Cygwin команду:
g++ -I /usr/local/systemc-2.2/include -L /usr/local/systemc-2.2/lib-cygwin method.cpp -lsystemc -o ccc.x
9.	Запустити виконавчий файл програми параметричного налаштування “ccc.x”. Результати параметричного налаштування будуть виведені в консоль Cygwin. 
За необхідності їх можна перенаправити в файл.
